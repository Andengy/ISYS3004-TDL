To Do List Webpage

This was our 4th webpage development project, where we would create a to do list. This webpage required HTML, CSS and Javascript in order to operate. The function was simple, it would allow the users to add into the text box where they would write a list of things to do. And would be able to check it off as they completed them.

This webpage allowed us to learn the language of HTML, CSS and Javascript, Hypertext Markup Language is where we would use HTML to input the text we want and adding links from there. Essentially HTML is used to define the content and structure of our webpage

While CSS, Cascading Style Sheet is a form of language used to edit the presentation, format, styling and effect to give it more character and not like a boring plain page. In this case we wanted a simple webpage where the idea of the webpage is located in the middle.

And finally Javascript, or also known as JS for short. This language would allow users to interact with the webpage for example within this webpage To Do List, javascript allowed us to create interaction between users and the webpage, if you finish a item on the list you would go to the checkbox and simple click it, this creates a chain reaction within the code where it would tick off the box and put a line through the item to show the users they have completed the item.
